# Gemini에게 질문하기

Google의 Gemini LLM에게 주제에 대한 관점을 문의합니다.

## 사용법

계획, 아키텍처 결정, 코드 검토 또는 다양한 AI 관점이 가치를 더하는 모든 상황에서 Gemini의 의견이 필요할 때 이 기술을 사용합니다.

## 지침

1. 관련 맥락을 포함하는 명확하고 구체적인 프롬프트를 작성합니다.
2. 프롬프트와 함께 아래 명령을 실행합니다.
3. Gemini의 응답을 구문 분석하고 비판적으로 평가합니다.
4. 유용한 통찰력을 자신의 추론에 통합합니다.

## 명령

```bash
gemini --prompt "<여기에 프롬프트 입력>"
```

## 가이드라인

- 항상 프롬프트에 충분한 맥락을 포함합니다 (프로젝트 세부 정보, 제약 조건, 이미 결정된 사항).
- 일반적인 조언이 아닌 실행 가능하고 구체적인 답변을 얻기 위해 질문을 구성합니다.
- Gemini의 응답이 자신의 분석과 충돌하는 경우, 의회 토론을 위해 불일치를 표시합니다.
- Gemini의 출력을 맹목적으로 수용하지 마십시오 — 의회의 한 목소리로 취급하십시오.
- 여러 부분으로 구성된 질문의 경우, 더 명확한 응답을 위해 별도의 호출로 나눕니다.
- 최상의 결과를 위해 프롬프트를 4000자 미만으로 유지합니다.
- 시간 초과: 응답에 최대 60초를 허용합니다.
